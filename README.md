#  Project Name : Bibliodrop

##  Purpose
Bibliodrop is a modern, responsive web application designed for library management and book browsing. It provides a seamless user experience with role-based access control, allowing standard users to browse and borrow books, while providing dedicated dashboards for Librarians and Administrators to manage the platform.

## Key Features
 **Role-Based Access Control (RBAC):** Custom dashboards and routing based on user roles (`admin`, `librarian`, `user`).
 **Secure Authentication:** Seamless login, registration, and session management using a custom auth client.
 **Responsive UI/UX:** Fully optimized navigation for both desktop and mobile devices using Tailwind CSS.
 **Modern Component Library:** Built with HeroUI for sleek, accessible components (Avatars, Spinners, Dropdowns).
 **Smart User Profiles:** Dynamic avatar fallback generation and state-driven profile dropdowns.
 **Dark Mode Ready:** UI prepared for theme toggling
